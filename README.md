# android-shopping-order

## UI
- [ ] 데이터가 로딩되기 전 상태에서는 스켈레톤 UI를 노출한다.
- [ ] 메인 액티비티가 로딩되기 전에 서버를 선택한다.

## Domain
- [ ] 사용자 인증 정보를 저장한다.
- [ ] 서버를 선택하여 교체할 수 있다.

___

## 주문하기 페이지
- [ ] 주문하기 버튼을 누르면 주문 상품 리스트가 띄워진다
- [ ] 사용 가능한 포인트가 보여진다
- [ ] 사용 할 포인트의 양을 적을 수 있다
- [ ] 결제 금액이 보여진다
- [ ] 주문하기 버튼을 클릭하면 서버에 해당 데이터가 올라간다.
- [ ] 주문 내역 화면으로 이동한다.
